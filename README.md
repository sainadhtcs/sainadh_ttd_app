# My To-Do List App

A simple, clean to-do list app. It runs in your browser and remembers your tasks even after you close the tab (using `localStorage`).

## Files
- `index.html` — the page structure
- `style.css` — the look and colors
- `script.js` — the logic (add, complete, delete, save)

## Run it on your own computer
Just double-click `index.html`, or open it with your browser.
